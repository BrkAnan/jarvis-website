◈ DEPLOYMENT

Initialize J.A.R.V.I.S

Get your personal AI assistant running in under 3 minutes.

01

Install Python 3.11 (64-bit)

Download and install Python 3.11 64-bit for Windows. Make sure to check "Add to PATH" during installation.

02

Get Your Gemini API Key

Visit Google AI Studio and generate a free API key. This powers JARVIS's intelligence.

03

Install Dependencies

cd C:\Path\To\JARVIS pip install -r requirements.txt
04

Configure API Key

Navigate to C:\Users\user\Downloads\JARVIS\config and open the api_keys.json file to enter your Gemini API key manually.

05

Launch JARVIS

py -3.11 main.py

JARVIS is now online and ready to assist.
